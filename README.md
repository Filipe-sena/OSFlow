# OSFlow 🛠️

Sistema de gestão de Ordens de Serviço (OS) desenvolvido para facilitar o controle de atendimentos técnicos.

## 🚀 Funcionalidades
- Cadastro de clientes e serviços.
- Listagem dinâmica de ordens.
- Armazenamento local (os dados não somem ao atualizar a página).
- Interface responsiva para uso em dispositivos móveis.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estrutura semântica)
- **CSS3** (Layout com Flexbox)
- **JavaScript Vanilla** (Lógica, manipulação do DOM e LocalStorage)

## 📚 Aprendizados (ADS)
Este projeto foi fundamental para consolidar conceitos de:
1. **Persistência de dados:** Uso do `localStorage` para simular um banco de dados.
2. **DOM:** Criação dinâmica de elementos em tela com JS.
3. **Versionamento:** Controle de versão completo utilizando Git e GitHub.
